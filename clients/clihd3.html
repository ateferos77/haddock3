Run HADDOCK3¶

Run HADDOCK3 docking simulation.

This is the main command-line client to run HADDOCK3 docking simulations. To prepare a simulation, setup configuration file defining a HADDOCK3 workflow and use this command-line client to execute that workflow.

Usage:

haddock3 -h
haddock3 <CONFIG FILE>

usage: haddock3 [-h] [--restart RESTART] [--extend-run EXTEND_RUN] [--setup]
                [--log-level {DEBUG,INFO,WARNING,ERROR,CRITICAL}] [-v]
                recipe

Positional Arguments¶

recipe

The input recipe file path

Named Arguments¶

--restart

Restart the run from a given step. Previous folders from the selected step onward will be deleted.

--extend-run

Start a run from a run directory previously prepared with the haddock3-copy CLI. Provide the run directory created with haddock3-copy CLI.

--setup

Only setup the run, do not execute

Default: False

--log-level

Possible choices: DEBUG, INFO, WARNING, ERROR, CRITICAL

Default: “INFO”

-v, --version

show version
