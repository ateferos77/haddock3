Defaults definitions

All default parameters used by the framework.

haddock.core.defaults.CNS_MODULES = ['rigidbody', 'flexref', 'emscoring', 'mdscoring', 'mdref', 'emref']

List of CNS modules available in HADDOCK3.

haddock.core.defaults.INTERACTIVE_RE_SUFFIX = 'interactive'

Suffix added to interactive haddock3-re runs.

haddock.core.defaults.MAX_NUM_MODULES = 10000

Temptative number of max allowed number of modules to execute

haddock.core.defaults.MODULE_DEFAULT_YAML = 'defaults.yaml'

Default name of the yaml default parameters file.

haddock.core.defaults.MODULE_IO_FILE = 'io.json'

Default name for exchange module information file

haddock.core.defaults.MODULE_PATH_NAME = 'step_'

Module input and generated data will be stored in folder starting by this prefix

haddock.core.defaults.RUNDIR = 'run_dir'

Default run directory name.
